# Feature 5: Click to Open

## Goal

Clicking a card opens the corresponding markdown file.

## What this does

- Clicking a card navigates to and opens the file that the card represents
- The file opens in the standard Obsidian editor

## Acceptance criteria

- Clicking a card opens the correct file
- The file opens in a new leaf / tab following Obsidian's default behavior

## Dependencies

- Feature 3 (card rendering)
